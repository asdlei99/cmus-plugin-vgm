# cmus-plugin-vgm
Video Game Music file support for cmus (vgm/vgz/dro/s98)

## Options

| Option name           | Value                                                                                                             |
| ----------------------| ----------------------------------------------------------------------------------------------------------------- |
| `input.vgm.max_loops` | Number of times a music will play its loop. It requires to clear the cache manually in order to update durations. |
